🎯 Пошаговый алгоритм выполнения работы:

  1. Подготовка данных: Откройте скачанный файл в Microsoft Excel. Ознакомьтесь с выборкой. Столбец X содержит объясняющую переменную (фактор), а столбец Y — зависимую переменную (результат).
  2. Активация инструментария: Перейдите на вкладку верхнего меню «Данные». Убедитесь, что в крайней правой панели у вас включен инструмент «Анализ данных».
    * Если его нет, зайдите в Файл -> Параметры -> Надстройки -> Надстройки Excel (Перейти) -> Отметьте галочкой «Пакет анализа».
  3. Запуск регрессионного анализа: Нажмите на «Анализ данных», выберите в списке строку «Регрессия» и нажмите ОК.
  4. Ввод диапазонов ячеек:
    • В поле «Входной интервал Y» выделите мышкой столбец с данными Y (включая заголовок).
    • В поле «Входной интервал X» выделите столбец с данными X (включая заголовок).
    • Обязательно поставьте галочку напротив пункта «Метки» (чтобы Excel учел названия столбцов).
  5. Вывод результатов: В параметрах вывода выберите пункт «Новый рабочий лист» и нажмите ОК.
  6. Анализ уравнения: На созданном листе найдите таблицу коэффициентов. Используя значения «Y-пересечение» (коэффициент $a$) и переменной X (коэффициент $b$), запишите итоговое уравнение парной регрессии: Y = a + b*X.
  7. Оценка качества: Посмотрите на значение показателя R-квадрат (R-square). Сделайте письменный вывод о том, на сколько процентов построенная модель объясняет изменчивость зависимой переменной.

Выполненную работу сохраните под именем ФИО_Эконометрика_ЛР2.xlsx и отправьте на проверку.
